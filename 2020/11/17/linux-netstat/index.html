简介

netstat 是控制台命令，是一个监控tcp/ip网络有用的工具， n用于显示各种网络相关信息， 如网络连接， 路由表，接口状态, 多播成员， masquerade连接等。一般用于检测本机端口的网络连接情况。

常用命令参数

参数 说明
-a 列出所有的连接
-t 列出所有的tcp连接
-u 列出所有的udp连接
-n 禁用域名解析
-l 仅列出有在 Listen (监听) 的服务状态。
-p 显示PID
-c 持续输出，定时执行netstat命令

检查网卡状态

netstat 能够列举出所有的网卡，或者看网卡详情，与此功能相同的命令为ip 或者 ethtool。

通过netstat -I 参数可以统计网卡的包统计情况，可用于网卡丢包排查

统计包信息

通过netstat -s 显示网络的统计信息，包括接收到的ip/icmp/tcp/udp包数，转发的ip/icmp/tcp/udp包数等

列举路由表

netstat能列出当前主机的路由表，曾几何时就知道查看路由表只有ip route 跟 route ,不曾想netstat也能列出路由表

netstat -r 用于列出当前主机的路由表
